## 一、八字排盘

### 1.1 输入参数

| 参数 | 格式 | 说明 |
|------|------|------|
| 出生日期 | YYYY-MM-DD | 阳历日期 |
| 出生时间 | 时辰 | 子/丑/寅/卯/辰/巳/午/未/申/酉/戌/亥 |
| 性别 | 男/女 | 用于后续分析 |

### 1.2 天干地支计算

**年柱**：以立春为界，查万年历确定当年干支

**月柱**（以节气为界）：
| 月份 | 节气 | 地支 |
|------|------|------|
| 正月 | 立春-惊蛰 | 寅 |
| 二月 | 惊蛰-清明 | 卯 |
| 三月 | 清明-立夏 | 辰 |
| 四月 | 立夏-芒种 | 巳 |
| 五月 | 芒种-小暑 | 午 |
| 六月 | 小暑-立秋 | 未 |
| 七月 | 立秋-白露 | 申 |
| 八月 | 白露-寒露 | 酉 |
| 九月 | 寒露-立冬 | 戌 |
| 十月 | 立冬-大雪 | 亥 |
| 十一月 | 大雪-小寒 | 子 |
| 十二月 | 小寒-立春 | 丑 |

月天干 = 根据年干推算（五虎遁诀）：
```
甲己之年丙作首，乙庚之岁戊为头
丙辛必定寻庚起，丁壬壬位顺行流
若问戊癸何方发，甲寅之上好追求
```

**日柱**：查万年历（或使用公式计算）

**时柱**（五鼠遁诀）：
```
甲己还加甲，乙庚丙作初
丙辛从戊起，丁壬庚子居
戊癸何方发，壬子是真途
```

## 二、日主五行分析

### 2.1 日主判定
取日干为日主，代表命主自身。

日干对应五行：
```
甲乙 → 木
丙丁 → 火
戊己 → 土
庚辛 → 金
壬癸 → 水
```

### 2.2 五行旺衰计分

统计八字四柱中每个五行的出现次数（天干+地支藏干），并加上月令权重：

| 位置 | 权重 |
|------|------|
| 月令（月支） | ×2 |
| 日支 | ×1.5 |
| 年支、时支 | ×1 |
| 天干 | ×1 |

### 2.3 五行喜忌判定

```
if 日主五行 > 平均值的1.5倍:  → 身强，喜克泄耗（忌生扶）
elif 日主五行 < 平均值的0.5倍: → 身弱，喜生扶（忌克泄耗）
else: → 中和，以平衡为要
```

**喜神**：对日主有利的五行（取名时优先选用该五行属性的字）
**忌神**：对日主不利的五行（取名时避免使用）

## 三、三才五格笔划分析

### 3.1 五格计算（以康熙字典笔划为准）

设姓氏笔划为 S，名1笔划为 M1，名2笔划为 M2：

| 格 | 计算方式 | 解释 |
|----|----------|------|
| 天格 | S + 1（单姓） | 祖运，先天运势 |
| 人格 | S + M1 | 主运，一生核心 |
| 地格 | M1 + M2（双字名） | 前运，36岁前 |
| 外格 | M2 + 1 | 副运，人际关系 |
| 总格 | S + M1 + M2 | 后运，36岁后 |

### 3.2 笔划吉凶（基于81数理）

**大吉数（推荐）**：
1, 3, 5, 6, 7, 8, 11, 13, 15, 16, 17, 18, 21, 23, 24, 25, 29, 31, 32, 33, 35, 37, 39, 41, 45, 47, 48, 52, 57, 63, 65, 67, 68, 73, 81

**大凶数（避免）**：
2, 4, 9, 10, 12, 14, 19, 20, 22, 26, 27, 28, 30, 34, 36, 40, 42, 43, 44, 46, 49, 50, 51, 53, 54, 56, 58, 59, 60, 62, 64, 66, 69, 70, 71, 72, 74, 75, 76, 77, 78, 79, 80

### 3.3 三才配置吉凶

三才 = 天格五行 · 人格五行 · 地格五行（按五行相生相克判断）

最优配置：**相生链**（木火土、火土金、土金水、金水木、水木火）

## 五、音韵美学评分

### 5.1 声调搭配规则

普通话四声：阴平(1) 阳平(2) 上声(3) 去声(4)

**推荐搭配**：
- 抑扬顿挫型：2+4、1+4、3+4
- 朗朗上口型：2+2、1+2、4+2
- 温婉柔和型：1+1、2+3

**避免搭配**：
- 3+3（拗口）
- 4+4（生硬）

### 5.2 谐音避讳检查

需检查的常见不雅谐音类别：
- 贬义词（如"范统"→"饭桶"）
- 动物名（如"朱"→ "猪"）
- 生理词汇
- 负面成语

## 六、AI 起名流程总览

```
输入(姓氏+生日+时辰+性别)
        │
        ▼
  [八字排盘] → 年柱/月柱/日柱/时柱
        │
        ▼
  [五行分析] → 日主强弱 + 喜神/忌神
        │
        ▼
  [生肖分析] → 喜用字根 / 忌用字根
        │
        ▼
  [候选字库] ← 按五行属性分类的汉字库
  [按喜神筛选] → 五行属性匹配喜神的字
  [按生肖筛选] → 字根符合生肖喜用的字
        │
        ▼
  [名字组合] → 姓氏 + 候选字1 + 候选字2
        │
        ▼
  [五格计算] → 五格笔划数
  [笔划筛查] → 剔除含凶数的组合
  [三才筛查] → 剔除三才凶配置
  [音韵评分] → 声调+谐音评分
  [谐音筛查] → 剔除不雅谐音
        │
        ▼
  [综合排序] → 按评分从高到低
        │
        ▼
  [输出结果] → Top N 个名字 + 分析报告
```

## 八、评分权重体系

| 维度 | 权重 | 说明 |
|------|------|------|
| 五行匹配度 | 30% | 名字五行是否符合喜神 |
| 五格吉凶 | 25% | 五格中吉数占比 |
| 三才配置 | 15% | 天地人三才是否相生 |
| 音韵好听度 | 15% | 声调平仄搭配 |
| 生肖契合度 | 10% | 字根是否符合生肖喜用 |
| 寓意美好度 | 5% | 字义是否积极正面 |

**最终评分** = Σ(各维度得分 × 权重) × 100

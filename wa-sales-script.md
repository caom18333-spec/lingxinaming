## 九、关键原则

| 原则 | 怎么做 |
|------|--------|
| **回复速度** | 越快越好，5分钟内回复最理想 |
| **不要硬推** | 客户犹豫就放一放，过半天再轻轻问一句 |
| **多用Emoji** | 但不要太多，每段1-2个刚好 |
| **语音消息** | 偶尔发一条10秒语音，比打字更有信任感 |
| **不要像机器人** | 偶尔打错字也没关系，更像真人 |
| **八字的免费诊断** | 永远免费先给一点东西，客户才愿意付费 |
